Central hub — Kitchen Dashboard
New modules to build (Planned)
Existing tools (Live)
Data flow direction
Dashboard ↔ module links
The pipeline: Supplier invoices land in the accounts email → the Invoicing Tool parses and categorises them → Monthly Spend rolls them up into kitchen product costs → GP & Recipe Costing uses live ingredient prices to calculate dish margins → Prep System pulls par levels and recipe yields to generate daily prep lists. All modules are accessible from — and feed data back into — the central Kitchen Dashboard.
